# Архитектура ПТО.ИИ

## Назначение

ПТО.ИИ — интеллектуальная платформа для инженеров ПТО, руководителей проектов и строительных организаций.

## Основные модули

- Telegram-бот
- Веб-интерфейс
- Голосовой помощник
- RAG-база знаний
- Анализ проектной документации
- Анализ исполнительной документации
- BIM-поиск
- Управление закупками
- Автоматизация исполнительной документации

## Используемые технологии

- Python
- FastAPI
- Telegram Bot API
- Ollama
- ChromaDB
- GitHub
- Tilda
- Yandex Cloud

## Цель проекта

Создать интеллектуального помощника, который понимает строительную документацию и помогает инженеру принимать решения за секунды.
